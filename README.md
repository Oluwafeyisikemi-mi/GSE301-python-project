# Student Academic Performance Analysis System

## Project Overview
This is a Python-based system designed to store, process, and analyze academic data for students in the Agriculture Department. It satisfies the requirements for the GSE301 Data Science project.

## Features
* **Data Management**: Stores student profiles using dictionaries and lists.
* **Grading System**: Automatically calculates letter grades and provides feedback.
* **Input Validation**: Uses Try-Except blocks to ensure age and CGPA inputs are valid.
* **Interactive Menu**: A console-based interface for easy navigation.
* **Eligibility Checker**: Determines graduation status based on CGPA and active status.

## How to Run
1. Ensure you have Python installed.
2. Run the script using the command: `python student_analysis.py`

## Technologies Used
* Python 3.x
* Data Structures: Lists, Tuples, Dictionaries, Sets
